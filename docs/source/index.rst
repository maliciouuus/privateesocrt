Introduction
------------

EscortDollars est une plateforme de parrainage innovante conçue pour les escortes et les ambassadeurs. 
Cette plateforme permet de maximiser les revenus grâce à un système de parrainage sophistiqué et 
des outils marketing puissants.

Principales fonctionnalités
--------------------------

* Système de parrainage multi-niveaux
* Solution White Label
* Tableau de bord analytique en temps réel
* Système de paiement sécurisé
* Interface utilisateur moderne et intuitive
* Support multilingue

Installation rapide
-----------------

.. code-block:: bash

   git clone https://github.com/yourusername/escortdollars.git
   cd escortdollars
   python -m venv venv
   source venv/bin/activate  # Sur Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python manage.py migrate
   python manage.py runserver

Pour plus de détails sur l'installation et la configuration, consultez la section :doc:`installation`.
